# Space Auto Tech - Quick Start Guide

## Getting Started

### Prerequisites
- Node.js 18+ installed
- npm or yarn package manager
- A modern web browser

### Installation

1. **Navigate to the project directory**:
   ```bash
   cd space
   ```

2. **Install dependencies**:
   ```bash
   npm install --legacy-peer-deps
   ```
   Note: Using `--legacy-peer-deps` due to React 19 RC compatibility

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser**:
   Navigate to [http://localhost:3000](http://localhost:3000)

## Available Scripts

### Development
```bash
npm run dev
```
Runs the app in development mode with hot reloading.

### Build
```bash
npm run build
```
Creates an optimized production build in the `out` directory.

### Production
```bash
npm run start
```
Starts the production server (run `npm run build` first).

### Lint
```bash
npm run lint
```
Checks code quality and potential errors.

## Customization

### Colors
Edit `tailwind.config.ts` to customize the color scheme:
```typescript
colors: {
  primary: { /* your colors */ },
  secondary: { /* your colors */ },
}
```

### Fonts
Modify `app/globals.css` to change fonts:
```css
@import url('your-google-fonts-url');
```

### Content
- **Services**: Edit `app/components/home/page.jsx`
- **Products**: Update data in `app/store/page.jsx`
- **Stock**: Modify inventory in `app/stock/page.jsx`

## Common Tasks

### Adding a New Page
1. Create a new directory in `app/`
2. Add a `page.jsx` or `page.tsx` file
3. The route will be auto-generated by Next.js

### Updating the Navbar
Edit `app/component/Navbar.js` and modify the `navItems` array:
```javascript
const navItems = [
  { name: "Your Page", href: "/your-route" },
  // ...
];
```

### Changing Images
Replace images in the `public/` directory, maintaining the same file names, or update image paths in components.

## Troubleshooting

### Port Already in Use
```bash
# Kill the process on port 3000
npx kill-port 3000
# Or use a different port
npm run dev -- -p 3001
```

### Build Errors
```bash
# Clean install
rm -rf node_modules package-lock.json
npm install --legacy-peer-deps
```

### Image Loading Issues
- Ensure images exist in `public/` directory
- Check file paths are correct
- Verify image formats are supported (jpg, png, webp, etc.)
